  1. Update Prices Page

    2. Make sure the Prices page is up to date. The system uses this page to lookup the prices to use when adding up an instructors paycheck. Make sure the prices are spelled exactly as they are in MindBody.

  2. Update Classes Page

    3. Check that the Classes page is up to date. This page is used by the system to include the name of each class that is listed on an instructors paystub.

  3. Export Payroll Report from MindBody

    • On MindBody, go to Reports
    • Under 'Staff', go to 'Payroll'
    • Make sure 'All Locations' is selected under the locations dropdown
    • Select the following 4 Options under the 'Service Category' area on the right
      1. Dance Classes
      2. Fitness Classes
      3. Private Lessons
      4. Workshops
    • Select the correct Pay Period Dates
    • Click the 'Generate' Button
    • Once the Report shows on screen, click the 'XLS' icon next to the 'Generate' button
    • Save the file to your computer. Do not change the file name.

  4. Upload Page

    5. Back on the VMAC Payroll website, go to the Upload page and upload the file from MindBody.
    This creates all of the Paystubs. Once finished, it will redirect you to the 'Paystubs' page. This can take up to 1 minute to complete.

  5. Check the Unpaid page

    6. The Unpaid page lists any classes with Price options that could not be found on the 'Prices' page.
    This page makes it easy to see any outlying classes that may have been entered into MindBody incorrectly.
    This also makes it easy to see if the Prices page is missing a pricing option.

  6. Export Paystubs

    7. Back on the Paystubs page, click the export page at the top of the screen. This will output a downloadble zip folder with all of the paystubs for the pay period. This process can take up to 1 minute.
